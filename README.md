## ParallaxListView
===================================      
防QQ图片墙，下拉带视差效果的listview    

在build.gradles添加:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

在dependencies添加:

	dependencies {
	        compile 'com.github.dangyao:ParallaxListView:-SNAPSHOT'
	}
